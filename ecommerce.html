TREND HUNTER

E-commerce UI Kit

The Trend Hunter Shop is a flexible and multipurpose mobile UI kit designed for e-commerce use. It has a complete set of views that cover multiple use cases and was made to give you the ability to generate new views using its own components. 100%vector, customizable and pixel perfect. Of course, it was made in Sketch 4.
