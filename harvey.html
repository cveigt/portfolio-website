Harvey was a weekend design exercise. I came with the idea of a daily motivational app. The name was taken from one of the main characters of USA Network's TV show "Suits".

The goal was to create a mobile app that notifies users with a daily motivational quote. Additionally, the app provides resources pulled from different sources.

Scope: Develop the visual language and create the main screens in one weekend. It had to be elegant, clear, and simple to use. I thought about making a working prototype in Swift but it never happened :)
